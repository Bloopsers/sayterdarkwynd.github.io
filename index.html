  • Got a bug?

    Report all bugs using your starbound.log file. It can be found in Starbound/storage/ under the filename starbound.log. If you have file extensions disabled, you will want to enable them.

    Paste the contents within a page at Pastebin.com and send it along.


    If you do not attach an error report, your bug will be ignored.

  • Mod Features

    FrackinUniverse is not a small mod by any means. Expect a learning curve. The following is a general list of features and capabilities.

    + Over 20 new major biomes

    + Over 70 sub-biomes

    + Over 500 new armors, techs, weapons and gear

    + Hundreds of craftable decorations

    + Hundreds of new craftable resources

    + 100+ new plants, each with their own useful buffs

    + Complex gene-engineering system for creating new plants

    + Advanced wired stations

    + 20+ new liquids

    + 200+ new tiles

    + Huge storage containers

    + 150+ new dungeons, microdungeons and missions

    + New physics

    + Tile-based effects (snow is slippery, etc)

    + Liquid and Solid tile interactions

    + Over a dozen new ores

    + New star types

    + New weather types

    + Temperature-based habitable zones

    + Over 100 new tutorial quests

    + Bees! mod included

    + Starbooze mod included
